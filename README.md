# HaxBevy

HaxBevy is a Rust clone of the game [HaxBall](https://www.haxball.com/) developed with the [Bevy](https://bevyengine.org/) game engine.

The game is available on itch.io: [HaxBevy](https://wazarr.itch.io/haxbevy)
